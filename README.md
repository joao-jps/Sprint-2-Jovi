## 🎯 Visão Geral

**Jovi Vision Core** é um protótipo inovador de aplicação web que apresenta uma câmera inteligente acadêmica. Desenvolvida especificamente para estudantes, oferece qualidade profissional de imagem para anotações, apresentações e conexões sociais — tudo isso acessível diretamente pelo celular.

Este projeto combina design moderno, funcionalidade interativa e uma arquitetura frontend bem estruturada para criar uma experiência de usuário excepcional.

---

- 🔐 **Autenticação de Usuário** - Sistema de login com validação segura
- 📋 **Formulário de Avaliação** - Coleta estruturada de feedback do usuário
